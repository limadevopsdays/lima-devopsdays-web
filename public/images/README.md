## ⚙️ Instrucciones de Exportación

1. **Descarga las imágenes originales** desde Figma Make antes de exportar
2. **Renombra los archivos** según la tabla anterior
3. **Coloca las imágenes** en las carpetas correspondientes
4. Las rutas ya están configuradas en el código para funcionar automáticamente

---

**Nota**: Si exportas desde Figma Make, este proceso debería ser automático. Si copias el código manualmente, necesitarás descargar y colocar las imágenes manualmente.
